This code is a comprehensive steering effort calculator for an FSAE vehicle for use in design presentations. One of the largest engineering challenges any team faces is being able to display and present data in a useful way to others. This code combines a strong physics model alongside advanced plotting features to perform a baseline assesment of steering effort as well as a sensistivity analysis.

To run the code, download the repo and install all packages. I have included some preset vehicle configurations, but upon running the Effort_Calculator.py file, a screen will pop up where you can change values. You can also select other static config files for comparison with the current setup.

Once ready, press the calculate button to open the plots. Each plotting screen (the overall summary and sensitivity screens) have multiple plots you can hover over and explore in more detail with an auto-snapping cursor. Additionally, you can use the controls in the bottom left of the screen to adjust the plots.

Run the calculator for any number of steups and download the csv files generated for the sweeps! Enjoy!
